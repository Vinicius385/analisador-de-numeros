# Analisador de Números

Aplicação web desenvolvida para análise de números utilizando HTML, CSS e JavaScript.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- Adição de números
- Validação de valores
- Análise estatística
- Soma dos valores
- Maior e menor número
- Média dos números inseridos
